## Задание

Реализовать функциональный класс, использующий алгоритм Хаффмана для кодирования символов в заданной строке. 
Возможности предоставляемые пользователю:
	- Построение H-дерева из переданной последовательности символов;
	- Представление любой последовательности этих символов в виде бинарного кода;
	- Декодирование бинарного кода в последовательность символов, согласно построенному дереву;
	- Получение соответствия символов и бинарных кодов.

Все необходимые математические сведения можно найти в статье
["Код Хаффмана"][Huffman coding] на сайте Wikipedia.
